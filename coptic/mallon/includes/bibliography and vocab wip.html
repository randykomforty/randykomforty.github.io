(Bulletin de l'Ac. imp. des Sciences de St-Pétersbourg: I–IX dans le tome X, N° 5, 1899; X–XX dans le tome XIII, N° 1, 1900; XX–XXV dans le tome XIV, N° 3, 1901; XXVI–XLV dans le tome XXI, N°s 3 et 5, 1904). — Quatre tirages à part.

Dictionary editor

Editor for building new entries for etymological dictionaries.

Key:

  • _[(text)] = italics
  • *[(text)] = bold
  • _d[(text)] = dialect
  • ~co1[(text)] = headword
  • ~co[(text)] = Coptic text
  • ~gr[(text)] = Greek text
  • ~ab[(text)] = Arabic text
  • ~he[(text)] = Hebrew text
  • ~am[(text)] = Aramaic text

Example syntax:

~co1[ϩⲟⲩⲏⲧ] _d[S]_d[A2], pl ~co[ϩⲟⲩⲁⲧⲉ] _d[S] & sg as pl, nn, _[passenger] on board ship (?): Ex 27 29 ~co[ⲛⲉϩ.] (_d[B] om) ~gr[ἐπιβάτης] (var ~gr[ναύτης]) _[nauta]; P1312 156 ~co[ⲛⲛⲉⲉϥ…ⲙⲛⲛⲉϩ. ⲉⲧⲧⲁⲗⲏⲩ ⲛⲙⲙⲁⲩ]
